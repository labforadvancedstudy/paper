# Executive Function

CEO of the mind.

Core components:
- Inhibition: Stop that impulse
- Shifting: Switch tasks
- Updating: Refresh information

Prefrontal cortex headquarters.
Last to develop (age 25).
First to decline (aging).

Impaired in:
- ADHD
- Depression
- Sleep deprivation
- Stress

Trainable through practice.
