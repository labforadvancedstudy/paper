## 개념 (Concept)

계산은 상태의 변환이다. 정보가 규칙에 따라 다른 정보로 바뀌는 과정. 하지만 더 깊이 들어가면, 계산은 우주가 스스로를 이해하는 방식일지도 모른다.

## 원리 (Principle)

```
Universe as Computer:
- 모든 물리 과정은 정보 처리
- 양자역학적 상태 변화 = 계산
- 시공간 자체가 거대한 연산 기판

Computation Hierarchy:
1. 물리적 계산 (입자 상호작용)
2. 화학적 계산 (분자 반응)
3. 생물학적 계산 (신경망)
4. 디지털 계산 (컴퓨터)
5. 의식적 계산 (사고)
```

```python

def add(a, b):
    return a + b

# 실제로 일어나는 일
# 1. 전자의 흐름이 트랜지스터를 통과
# 2. 양자 터널링 효과로 상태 전환

```
