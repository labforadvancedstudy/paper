# Neuron Structure

Information processing cell.

Parts:
- Dendrites: Input receivers
- Soma: Processing center
- Axon: Output cable
- Terminals: Output transmitters

100 billion in human brain.
Each connects to ~7000 others.

Evolved from:
Simple ion channels → 
Excitable membranes →
Specialized cells

First neurons: ~600 Mya
Jellyfish, hydra
